# Log Retrieval Query

The following Kusto Query Language (KQL) statement was used to retrieve Azure Firewall network rule events from Log Analytics:

```kql id="9l91sq"
AzureDiagnostics
| where Category == "AzureFirewallNetworkRule"
| project TimeGenerated, msg_s, Action_s
| order by TimeGenerated desc
| take 50
```

### Query Purpose

This query filters Azure Firewall network-rule records, selects key event attributes, arranges the results in reverse chronological order, and displays the fifty most recent entries for analysis and troubleshooting purposes.

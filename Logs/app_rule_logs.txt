## Log Analytics Query

The following Kusto Query Language (KQL) statement was executed to retrieve and analyze recent Azure Firewall application rule events:

```kql
AzureDiagnostics
| where Category == "AzureFirewallApplicationRule"
| project TimeGenerated, msg_s, Action_s
| extend DestinationHost = extract("FQDN: ([^ ]+)", 1, msg_s)
| order by TimeGenerated desc
| take 50
```

This query filters Azure Firewall application-rule logs, extracts destination hostnames from the message field, sorts the results by event time, and returns the fifty most recent records for review.
